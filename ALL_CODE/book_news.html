Book instructions

The Project Book is a detailed document that describes the project thoroughly, including clear explanations and illustrative images. The book consists of multiple sections, carefully organized to showcase the work.

  • Cover Design:
    • First-Year Teams: The cover must be brown.
    • Second-Year Teams: The cover must be navy blue.
    • The cover must include the team members' names and the project supervisor's name.
  • The book contains clear descriptions, a breakdown of project sections, and illustrative images to clarify the work.
  • The project must be reviewed and approved by the supervisor before printing.
  • The total page count and number of sections must be clearly mentioned.
